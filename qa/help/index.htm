QuickAccess Hilfe

Startseite

Räume, Gewerke

Die Icons verzweigen in die Kanalliste. Dort werden Stati angezeigt, und Aktoren können geschaltet werden.

Software/Hardware

Programme: Liste aller Programme. Programme können zum Ausführen angeklickt werden.

Systemvariable: Liste aller Systemvariable mit Anzeige der aktuellen Werte. Systemvariable können angeklickt werden, um Details anzuzeigen und Werte zu ändern.

DirectAccess: Schnellzugriff auf häufig benötigte Systemvariable. Der DirectAccess-Zugriff kann über die jeweilige Systemvariable konfiguriert werden.

Servicemeldungen: Anzeige offener Servicemeldungen. Die Anzahl der Servicemeldungen wird bei jedem Aufruf der Startseite aktualisiert und dann zusammengefaßt angezeigt.

Einstellungen

Alle Einstellungen werden in der URL gespeichert. Wird diese URL in den Favoriten gespeichert, so startet QuickAccess direkt mit den gewählten Einstellungen. Jede Seite kann als Favorit gespeichert werden.

"Buttons kleiner" bzw. "Buttons größer": Mit diesen Buttons kann die Darstellungsgröße gewählt werden, um QuickAccess für verschiedene Bildschirmgrößen anzupassen.

Link für Lesezeichen: Dieser Link enthält die QuickAccess-Konfiguration (Spaltenanzahl usw.) und kann in die Favoriten augenommen werden. Damit ist die Optik bei jedem Aufruf voreingestellt.

Aktualisierung erzwingen: Bei Aktivierung stellt QuickAccess sicher, daß die Seite bei jedem Aufruf neu geladen wird, um den aktuellen Status anzuzeigen. Bei deaktivierter Aktualisierung kann es vorkommen, daß der Browser die letzte Version aus dem Cache anzeigt, statt die Seite von der CCU neu anzufordern.

PIN: Über diesen Button kann eine einfache PIN-Abfrage (Kindersicherung) eingestellt werden.

Installation als App

QuickAccess kann als Favorit gespeichert oder als App auf dem Home Screen von Mobilgeräten abgelegt werden.

Die Einstellungen der App (Button-Größe, Aktualisierung, Tablet-Ansicht usw.) werden in der URL gespeichert. Am sinnvollsten ist es daher, zunächst auf der Startseite die korrekten Einstellungen vorzunehmen:

  • "Buttons kleiner" und "Buttons größer" stellen die gewünschte Button-Größe ein
  • "Breite Darstellung" für übersichtlichere Darstellung auf Querformat-Bildschirmen aktivieren
  • "Aktualisierung erzwingen" wird bei manchen Browsern benötigt, die sonst bei der Rückkehr zu bereits aufgerufenen Seiten eine alte Version aus dem Cache wieder anzeigen
  • "App" aktivieren, damit QuickAccess vom Home Screen aus im Vollbild-Modus (ohne Navigationsleiste) ausgeführt wird
  • weil auch die Position auf der Seite in der URL gespeichert wird, zum Abschluss auf den Aufwärts-Pfeil in einer Überschrift klicken

Danach kann QuickAccess als Favorit gespeichert oder zum Home Screen hinzugefügt werden und startet in der gewählten Einstellung.
