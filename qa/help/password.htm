Kinder­sicherung

Die PIN-Abfrage ist ausschließlich als einfache Kindersicherung gedacht. Sie können durch eine mehrstellige Ziffernfolge den Zugriff auf QuickAccess sperren. Sie müssen zur Bestätigung der Eingabe den "OK"-Button betätigen. Die Bestätigung über [RETURN] auf der Tastatur ist nicht ausreichend.

Die PIN wird während der Nutzung von QuickAccess verschlüsselt in der URL übertragen. Diese Verschlüsselung ist sehr oberflächlich und ersetzt keine Authentifizierung, z.B, bei Nutzung über das Internet. Zudem werden von QuickAccess keinerlei Daten vor der Übertragung verschlüsselt.

Nutzen Sie Dienste wie z.B. meine-homematic.de, um aus dem Internet sicher auf Ihre CCU zugreifen zu können.

Kennwort vergessen

Um das Kennwort zurückzusetzen, löschen Sie die Systemvariable "QuickAccess PIN" über die WebUI.

Sicherheits­hinweise

Die neue PIN wird im Klartext in der URL angezeigt. Besucher, die Zugriff auf den Verlauf Ihres Browsers haben, können die PIN leicht auslesen. Löschen Sie daher den Verlauf, wenn Sie eine neue PIN vergeben haben, und schließen Sie das Browserfenster.

Wenn bei gesetzter PIN eine Seite aufgerufen wird, springt QuickAccess automatisch zur PIN-Eingabe. Um unnötige Ladezeiten zu vermeiden, setzen Sie einen Bookmark auf die Paßwortabfrage wenn der Hinweis erscheint "QuickAccess ist gesperrt".

Es gibt nur eine Session. Wenn Sie sich an mehreren Rechnern in QuickAccess anmelden, werden Sie an allen Rechnern abgemeldet, wenn Sie sich n nur einem der Rechner abmelden.
